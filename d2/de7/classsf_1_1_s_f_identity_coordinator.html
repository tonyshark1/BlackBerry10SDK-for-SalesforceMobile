Detailed Description

The SFIdentityCoordinator class is used to retrieve identity data from the ID endpoint of the Salesforce service. This class is not meant to be used by the SDK classes only.

The SFIdentityCoordinator class is used to retrieve identity data from the ID endpoint of the Salesforce service. This data will be based on the requesting user, and the OAuth app credentials he/she is using to request this information. This class is not meant to be used by the SDK classes only.
